# x-enhancements (x-form, x-password, x-tags, x-file)

These are helper attributes for form and input enhancements. See [src/wb-viewmodels/enhancements.js](../../src/wb-viewmodels/enhancements.js).

- **Type:** Attribute
- **Usage:** `[x-form]`, `[x-password]`, `[x-tags]`, `[x-file]`

## Description
Provides enhanced behaviors for forms and inputs, such as AJAX forms, password fields, tag inputs, and file uploads. Add the relevant `x-*` attribute to enable the enhancement.
