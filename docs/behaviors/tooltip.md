# x-tooltip Behavior

Shows a tooltip on hover/focus for any element.

- **Type:** Modifier
- **Usage:**
  ```html
  <button x-tooltip="Tooltip text">Hover me</button>
  ```
- **Note:** Use for contextual, hover-based help.
