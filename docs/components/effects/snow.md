# Snow

A falling snow effect.

## Usage

```html
<div data-wb="snow" data-count="30"></div>
```

## Properties

| Property | Attribute | Type | Default | Description |
|----------|-----------|------|---------|-------------|
| count | data-count | number | 30 | Number of snowflakes |
| showButtons | data-show-buttons | boolean | true | Show trigger button |
| repeat | data-repeat | boolean | true | Loop animation |
| delay | data-delay | string | "0s" | Start delay |
| duration | data-duration | string | "8s" | Animation duration |
