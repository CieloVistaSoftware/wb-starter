# Confetti

A colorful confetti explosion effect.

## Usage

```html
<div data-wb="confetti" data-count="50" data-label="Celebrate!"></div>
```

## Properties

| Property | Attribute | Type | Default | Description |
|----------|-----------|------|---------|-------------|
| count | data-count | number | 50 | Number of particles |
| label | data-label | string | "Fire Confetti!" | Button text |
| showButtons | data-show-buttons | boolean | true | Show trigger button |
| repeat | data-repeat | boolean | false | Loop animation |
| delay | data-delay | string | "0s" | Start delay |
| duration | data-duration | string | "3s" | Animation duration |
