# Fireworks

A fireworks burst effect.

## Usage

```html
<div data-wb="fireworks" data-count="30"></div>
```

## Properties

| Property | Attribute | Type | Default | Description |
|----------|-----------|------|---------|-------------|
| count | data-count | number | 30 | Number of particles |
| showButtons | data-show-buttons | boolean | true | Show trigger button |
| repeat | data-repeat | boolean | false | Loop animation |
| delay | data-delay | string | "0s" | Start delay |
| duration | data-duration | string | "1.5s" | Animation duration |
