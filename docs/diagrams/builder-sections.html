üìã Builder Rules

‚úèÔ∏è

Content = Canvas (WYSIWYG)

All text content (title, subtitle, content, footer) is edited directly on the canvas.
Click any text to edit it in place. What you see is what you get.

‚öôÔ∏è

Settings = Properties Panel

Non-visual settings (elevated, clickable, variant, href, columns) go in the panel.
These are checkboxes, dropdowns, and inputs ‚Äî not text content.

üì¶

Component Hierarchy

Layout containers (Container, Grid, Columns) hold Components (Cards, Buttons, etc.).
Components can also be placed at root level for simple pages.

üé®

Theme Inheritance

Default (inherit) ‚Üí Page uses site-wide theme from <html data-theme>
Override ‚Üí Page sets its own theme, stored as "theme": "dark" in page JSON
