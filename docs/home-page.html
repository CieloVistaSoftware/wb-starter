Home Page Documentation

Overview

The home page is the main entry point for the WB-Starter project. It showcases the core features, stats, live demo, and premium components. All content and behaviors are validated against the home-page schema.

Sections

  • Hero: Uses <wb-cardhero>, variant cosmic, headline, subtitle, CTA, accessible (role="banner")
  • Stats Banner: Four stat items: Behaviors, DOM Architecture, Build Time, Standards Compliant. No zero values allowed.
  • Live Demo: Interactive demo using <wb-demo> (ripple, tooltip, draggable, confetti)
  • Feature Cards: Six cards with icon, title, description. Accessible (role="region")
  • Premium Showcase: Audio visualization and smart notifications
  • Call to Action: Get started or view on GitHub

Schema Compliance

Validated by src/wb-models/home-page.schema.json. All required children, styles, and accessibility features enforced.

Testing

See tests/behaviors/ui/home-page-permutation.spec.ts. Stat values reset and checked dynamically.

Notes

  • No zero values permitted in stats or demo
  • All sections must render and be accessible

For further details, see the schema and test files referenced above.
