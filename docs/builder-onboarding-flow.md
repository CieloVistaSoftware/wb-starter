# Builder Onboarding Flow

> Describes the first-time user onboarding experience for the Visual Page Builder.

## Overview

When a user opens the builder for the first time, a guided onboarding flow walks them through the key features.

## Steps

1. **Welcome screen** — brief overview of the builder
2. **Canvas tour** — highlights the canvas, component palette, and property panel
3. **First component** — prompts the user to drag a `<wb-hero>` onto the canvas
4. **Property editing** — demonstrates changing the hero title in the property panel
5. **Save & preview** — shows how to save the page and preview it in the browser

## Skipping Onboarding

Users can skip onboarding at any step. Onboarding completion is stored in `localStorage`.
