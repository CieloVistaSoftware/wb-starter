# Builder Interaction Rules

> Rules governing how users interact with the Visual Page Builder.

## Canvas Interactions

- Click a component to select it and open its property panel
- Drag components to reorder within a section
- Double-click text components to edit inline

## Property Panel

- Properties are driven by the component's `.schema.json` file
- Required properties are shown first
- Boolean properties render as toggles
- Enum properties render as dropdowns

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Delete` | Remove selected component |
| `Escape` | Deselect |

> **Note:** Builder is currently not operational. These rules apply once it is restored.
