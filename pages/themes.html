How to Use Themes

Simple implementation with CSS custom properties

1. Set Theme Attribute

<html data-theme="ocean"> ... </html>

2. Switch via JavaScript

// Switch theme programmatically document.documentElement .setAttribute('data-theme', 'cyberpunk');

3. Use CSS Variables

.my-component { background: var(--bg-secondary); color: var(--text-primary); border: 1px solid var(--border-color); }
