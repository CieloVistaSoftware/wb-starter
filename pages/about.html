About Cielo Vista Software

The team behind wb-starter

Our Mission

Cielo Vista Software builds tools that make the web simpler to write and easier to maintain. Our flagship product, wb-starter, is a zero-build, schema-first web-component framework: you compose UIs from <wb-*> tags and x-* behaviors in plain HTML — no bundler, no build step, no framework lock-in.

Why wb-starter?

  • ✅ Zero build — open an HTML file and it just works
  • ✅ Composition over inheritance — stack x-* behaviors on any element
  • ✅ Light DOM, no Shadow DOM — inspect and theme everything
  • ✅ Schema-first — components are described in data, not hand-wired
  • ✅ Fully themeable — 20+ built-in themes, zero hardcoded colors
