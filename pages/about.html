WB Starter is a modern website template powered by WB Behaviors.

WB Starter is designed to get you up and running in seconds. No npm install, no webpack configuration, no waiting for builds. Just open index.html and start creating.

Why We Built This

Modern web development h5s become overly complex. We wanted to create something that:

  • Works out of the box with zero configuration
  • Uses semantic HTML that everyone understands
  • Provides beautiful themes without fighting CSS
  • Lets you focus on content, not tooling
