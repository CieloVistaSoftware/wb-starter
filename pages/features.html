✨ Features

Everything you need to build modern websites, with zero build step.

Why WB Starter?

No webpack, no npm install, no build process. Just open index.html and start editing.

Change your site structure in site.json. Navigation, footer, branding - all in one place.

23 beautiful themes with full CSS variable support. Switch themes without page reload.

Responsive layout that works on any screen size. Collapsible nav for mobile devices.

Works with standard HTML elements. No custom elements or Shadow DOM complexity.

Standard browser DevTools work perfectly. Inspect elements, edit styles, no magic.
