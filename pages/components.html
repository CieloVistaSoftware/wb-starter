🧩 Components Library

wb-starter v3.0 • Interactive demos with copy-ready code

Every component below is a real <wb-*> custom element — <wb-card>, <wb-progress>, <wb-modal>, and the rest. Drop the tag in, no build step, no base class to extend. And it all lives in Light DOM — no Shadow DOM, no encapsulation boundary — so your theme's CSS, your own selectors, and the browser's devtools reach straight into every component exactly the way they reach into plain HTML.
