Adding Pages

To add a new page:

  1. Create an HTML file in pages/ folder (e.g., pages/mypage.html)
  2. Add navigation entry in config/site.json:
    { "id": "mypage", "label": "My Page", "icon": "ğŸ“„", "page": "mypage" }
  3. Refresh the browser - your page is ready!

Using Behaviors

Add behaviors to any HTML element with data-wb attributes:

<!-- Simple behavior --> <button data-wb="ripple">Click me</button> <!-- With options --> <div data-wb="card" data-title="My Card" data-hoverable> Content here </div> <!-- Multiple behaviors --> <button data-wb="ripple tooltip" data-tooltip="Hello!"> Hover me </button>

Theming

Change the default theme in site.json:

"site": { "theme": "cyberpunk" // or any of 23 themes }
