What is a registered view?

A view is a reusable HTML template registered once under a name — badge, alert-box, file-toolbar — and then usable anywhere as its own custom tag. Change the template in one place and every instance on the page updates. Templates support {{attribute}} interpolation and wb-for/wb-if for lists and conditionals.

Single-word names get a wb- prefix as a tag (badge → <wb-badge>); hyphenated names are used as-is (alert-box → <alert-box>). Each view's real tag name is shown under its name in the table below.
