x-help Demo

When to use it

x-help is for hint text that a user should see without having to hover or interact — the kind of short instruction that sits permanently under a form field or next to a setting ("must be 8+ characters", "this can't be changed later"). If the explanation should only appear on hover/focus instead of taking up space all the time, use x-tooltip — the two behaviors are deliberately kept separate rather than one trying to do both jobs.

How to use it

Add the x-help attribute to any element (a <span>, a <div>, whatever fits your layout) whose text content is the help message. It adds consistent styling and accessibility (role="note" plus a wb-help class) — nothing else. If you don't need that consistency, a plain element works fine too; x-help exists so every hint in the project looks and reads the same way instead of each page inventing its own.
