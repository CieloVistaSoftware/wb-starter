Demonstrates the x-help behavior.
Note: x-help displays help text inline, always visible, and does not show on hover. For hover-based help, use x-tooltip instead.
Value: x-help is functionally the same as a <span> except it adds consistent styling and accessibility (like role="note" and a help class). If you don't need those, use a regular element instead.
