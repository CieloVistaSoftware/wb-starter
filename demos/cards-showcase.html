WB Behaviors is a lightweight, modular web component library built on native browser APIs. It provides a set of behaviors and custom elements that enhance your HTML without heavy dependencies.

To get started, simply include the WB bootstrap script and start using wb-* custom elements or x-* behavior attributes on your existing HTML.

The framework automatically discovers and initializes components, making it easy to progressively enhance your pages.

This card starts in the expanded state. Click 'Show Less' to collapse it.

You can toggle between states using the button below.
