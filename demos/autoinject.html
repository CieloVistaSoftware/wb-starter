10. Cards Enhanced

Value Added:

  • Semantic: Uses <article> tags automatically.
  • Structure: Automatically organizes header, main, and footer content.
  • Interactive: Supports hover effects and click actions with toast feedback.

Basic Card

This is a standard <article> tag. WB automatically enhances it into a card component.

Interactive Card

This card has hover effects and is clickable.

It also uses the primary variant.

Code:

```html <!-- Basic Card --> <article heading="Title" subtitle="Subtitle"> <p>Content...</p> </article> <!-- Interactive Card (shows toast on click) --> <article id="interactive-card" heading="Click Me" clickable="true" hoverable="true" variant="primary" tooltip="Card tooltip info"> <p>Content...</p> </article> <script> // Show toast instead of alert document.getElementById('interactive-card').addEventListener('click', () => { createToast('Clicked Interactive Card', 'success'); }); </script> ```

Available Decorators

Customize your cards with these data attributes:

Attribute Description Example
data-title Sets the card header title heading="My Card"
data-subtitle Sets the card header subtitle subtitle="Details"
data-clickable Enables click interactions clickable="true"
data-hoverable Enables hover elevation effects hoverable="true"
data-variant Applies a visual theme variant variant="primary"
data-tooltip Adds a tooltip (Implicit Behavior) tooltip="Info"
data-elevated Adds permanent shadow elevation elevated="true"
