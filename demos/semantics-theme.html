🎨 Theme & Dark Mode

Behaviors for managing themes and dark mode.

Theme Control wb-themecontrol

A dropdown to switch the global theme. Persists selection to localStorage.

Forced Theme x-darkmode

These sections have forced themes regardless of the global setting. target="self" scopes the theme to the element itself; theme="…" sets which theme (not data-theme, which x-darkmode doesn't read).

Nested Themes

Themes cascade through data-theme on any container — no behavior needed, just the plain HTML attribute the theme system itself keys off of. A nested container can override its ancestor's theme.
