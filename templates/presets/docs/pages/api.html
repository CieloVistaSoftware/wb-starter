API Reference

Complete reference for the REST API.

Base URL:

https://api.yourproject.com/v1

Authentication

All API requests require authentication via API key:

curl -H "Authorization: Bearer YOUR_API_KEY" \ https://api.yourproject.com/v1/users

Users

GET /users

List all users in your organization.

Parameters

limit integer Max results (default: 20)
offset integer Pagination offset

Response

{ "data": [ { "id": "usr_123", "name": "John Doe", "email": "john@example.com" } ], "total": 1 }
POST /users

Create a new user.

Body

{ "name": "Jane Doe", "email": "jane@example.com" }

Errors

The API uses standard HTTP status codes:

200 Success
400 Bad Request
401 Unauthorized
404 Not Found
429 Rate Limited
500 Server Error
